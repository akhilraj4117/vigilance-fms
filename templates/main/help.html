Help Guide

Getting Started

Welcome to the Vigilance File Management System! This application helps you manage vigilance-related files, disciplinary actions, RTI applications, and more.

User Roles:
  • Administrator: Full access to all features including user management, file deletion, and closing files.
  • Guest: Read-only access to view files and data. Can create new entries but cannot delete or close files.

Managing Files

Creating a New File:
  1. Go to Files → Create New File
  2. Enter the file number (format: YYYY/XXX/Category)
  3. Select the category and file type
  4. Fill in the subject and other details
  5. Click "Create File"
Closing a File (Admin only):
  1. Open the file you want to close
  2. Click the "Close File" button
  3. Enter closing remarks
  4. Confirm the action

Disciplinary Actions

Disciplinary actions are linked to files and track employee misconduct proceedings.

Creating a Disciplinary Action:
  1. Go to Disciplinary → New Action
  2. Enter the associated file number
  3. Enter employee PEN to auto-fill details
  4. Fill in action details (major/minor, MOC, SCN, etc.)
  5. Save the action

RTI Applications

Track RTI applications received and their processing status.

Creating an RTI Application:
  1. Go to RTI → New Application
  2. Enter the associated file number
  3. Fill in applicant details
  4. Enter subject and fee details
  5. Save the application
Managing Appeals:

RTI appeals (First Appeal, Second Appeal) can be created from the RTI application view page.

Employees

Maintain a master list of employees for reference in disciplinary actions.

Adding an Employee:
  1. Go to Master Data → Employees
  2. Click "Add Employee"
  3. Enter employee details (Name, PEN, Designation, etc.)
  4. Save the employee

Institutions

Maintain a master list of health institutions in the district.

Adding an Institution:
  1. Go to Master Data → Institutions
  2. Click "Create Institution"
  3. Enter institution details
  4. Save the institution

Reports

Generate various reports for analysis and record-keeping.

Available Reports:
  • File Summary - Overview of all files by status and category
  • Pending Files - List of all pending files
  • Closed Files - List of closed files with date range filter
  • Disciplinary Summary - Overview of disciplinary actions
  • RTI Summary - Overview of RTI applications
  • UA Report - Unauthorised absentees report
  • Court Cases - List of ongoing court cases

Most reports can be exported to CSV format for further analysis.

User Management

Note: Only administrators can manage users.

Adding a New User:
  1. Click on your username → Manage Users
  2. Click "Add New User"
  3. Enter username, password, and select role
  4. Click "Register User"
Changing Your Password:
  1. Click on your username → Change Password
  2. Enter current password and new password
  3. Click "Update Password"

Need More Help?
