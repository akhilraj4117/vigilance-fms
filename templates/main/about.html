About

Vigilance File Management System

Version 2.0 (Web)

Organization

District Medical Office (Health)
Thiruvananthapuram
Kerala, India

Description

The Vigilance File Management System is a comprehensive web-based application designed to manage and track vigilance-related files, disciplinary actions, RTI applications, court cases, and inquiries for the District Medical Office (Health), Thiruvananthapuram.

Features
  • File management with categorization and tracking
  • Disciplinary action management
  • RTI application processing and tracking
  • Court case management
  • Inquiry tracking
  • Employee and institution master data
  • Comprehensive reporting
  • Role-based access control
Technology

Built with Python Flask, SQLAlchemy, Bootstrap 5, and deployed on Render.com
