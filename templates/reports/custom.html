Help & Tips
How to use the Custom Report Builder:
  1. Select Data Source: Choose which type of data you want to report on.
  2. Choose Columns: Select which fields to include in your report.
  3. Set Date Range: Optionally filter by date range.
  4. Add Filters: Apply additional filters to narrow down results.
  5. Sort: Choose how to order your results.
  6. Generate: Click Generate to view your report.
Available Data Sources:
  • Files: Vigilance case files with categories, status, dates
  • Disciplinary: Disciplinary actions against employees
  • RTI: RTI applications and their processing status
  • Employees: Employee master data
  • Institutions: Institution details
  • UA: Unauthorised absentee records
Tip: For frequently used reports, consider using the pre-built reports from the Reports Dashboard for faster access.
