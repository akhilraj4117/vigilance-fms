Import Tips
Best Practices
  • Ensure PEN numbers are unique and correctly formatted
  • Use consistent date formats throughout the file
  • Check that institution names match existing records
  • Save CSV files with UTF-8 encoding
Common Issues
  • Duplicate PEN numbers will be skipped
  • Invalid dates will use NULL values
  • Empty required fields will cause row to be skipped
  • Extra columns in CSV are ignored
Processing
  • Large files may take several minutes
  • Import is done in batches of 100 rows
  • Failed rows don't affect successful imports
  • Retirement dates are auto-calculated from DOB if not provided
